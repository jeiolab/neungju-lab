## 2. 개인정보 수집 및 저장

### ✅ **안전**
- **로컬 스토리지**: `localStorage`, `sessionStorage` 사용 없음
- **쿠키**: 쿠키 사용 없음
- **서버 전송**: 사용자 데이터를 서버로 전송하는 코드 없음
- **데이터 저장**: 모든 게임 상태는 클라이언트 측 React `useState`에만 저장되며, 페이지 새로고침 시 자동 삭제됨

**저장되는 데이터:**
- 게임 점수, 레벨, 선택 이력 등 게임 진행 상태만 저장
- 개인 식별 정보(이름, 이메일, 전화번호 등) 수집 없음
- 사용자 입력(퀴즈 답안, 채팅 메시지)은 메모리에만 존재하며 서버로 전송되지 않음

## 3. XSS (Cross-Site Scripting) 취약점

### ✅ **안전**
- **dangerouslySetInnerHTML**: 사용 없음
- **innerHTML**: 사용 없음
- **eval()**: 사용 없음
- **document.write()**: 사용 없음

**사용자 입력 처리:**
- React의 기본 이스케이핑으로 안전하게 처리됨
- 마크다운 파싱은 제한적이며 (`**bold**`, `` `code` ``), HTML 태그는 렌더링되지 않음
- 모든 텍스트는 React의 JSX 텍스트 노드로 렌더링되어 자동 이스케이핑됨

**예시:**
```tsx
// ChatMessage.tsx - 안전한 렌더링
<div>{formattedText}</div>  // React가 자동으로 이스케이핑
```
